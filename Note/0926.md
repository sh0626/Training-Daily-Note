RAM = 정보 읽기,쓰기 가능 (power이 나가면 정보가 사라짐)

ROM = 읽기 전용 (power이 나가도 내부 정보가 젼하지 않음) (ex.USB, SSD 등등)

개발용 시스템(환경) = RAM 8GB(최소)

- DDR RAM (Double Data Rate RAM)
  
  - 정보를 줌과 동시에 받을 수 있는 RAM

정보 = 축적된 자료에서 의미있는 자료를 가공해 놓은 것

| 1byte | >>>>>>> | 8bit | 변환   |
| ----- | ------- | ---- | ---- |
| 0     | 0       | 6    | 110  |
| 1     | 1       | 7    | 111  |
| 2     | 10      | 8    | 1000 |
| 3     | 11      | 9    | 1001 |
| 4     | 100     | 10   | 1010 |
| 5     | 101     |      |      |

- 10진수 0~9까지
  
  - 8bit = 2^8 = 256
  
  - 256을 2진수로 나타내면 1 0000 0000

- 1byte
  
  - 1~255
  
  - 2^8 - 1 (8bit 안에 나타내기 위해 -1을 해준다)
  
  - -256 ~ 255

- BCD - 6bit

- ASCII - 7bit

- EBCDIC - 8bit

- UTF - 2Byte 문자코드
