- WAS
  
  - 처리서버
    
    - 웹서버의 질문을 처리
      
      - 결과를 DB에 전달
      
      - DB는 그에 맞는 결과를 다시 WAS로 보냄
      
      - WAS는 받은 값을 Web Sever로 전송
  
  - WAS 서버를 기동하기 위한 언어
    
    - PHP, JSP, sevlet
    
    - Spring, SpringBoot

- MVC 모델
  
  - 컨테이너 모델
  
  - DB / Web Server / Was가 합쳐져있음
  
  - Frame Work라 부름
  
  - 규격화된 처리 수단

- 동적 프레임워크
  
  - React
    
    - 자바스크립트 연동 및 성능 향상시킨 것
  
  - Vue
    
    - JSP, servlet, Spring, SpringBoot와 연동
