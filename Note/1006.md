- 자료구조

자료 >> 가공처리 >> 정보

정보처리과정 =>> 자료를 PC로 변형, 변조하여 의미있는 정보(결과)를 발생시키는 것



- 변조
  
  - A -> Digital

- 복조
  
  - D -> Analog



| 4bit  | word                    |
| ----- | ----------------------- |
| 6bit  | BCD code                |
| 7bit  | ASCII code              |
| 8bit  | EBCD Interger Code      |
| 16bit | UniCode (UTF-8 언어코드 체계) |



- 정보 보호 특징
  
  - 무한복제 가능 -> 유일성 확인불가
    
    - 유일성 확보를 위해 블록체인 기술 개발
      
      - SHA256
        
        - 256비트 암호화 체계 (ex.bitlocker)
