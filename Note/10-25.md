#### HTML

- hr 태그
  
  - 수평선 그어짐

- p 태그
  
  - 줄바꿈(Enter) + Block (한블럭으로 만듬)
  
  - 한 문단을 만든다고 보면 된다

- h 태그
  
  -  <h1>  부터 <h5>까지 있다
  
  - 글씨 강조, 굵기 변경

- &lt; 문자 &gt; 
  
  - 꺽쇠삽입
  
  - &lt + 세미콜론
  
  - &gt + 세미콜론

- br태그
  
  - 줄바꿈 (한줄만 Enter)

- width
  
  - 반응형 웹페이지 만들때 %로 값을 준다

- div 태그
  
  - 개별적으로 css를 먹일 수 있다
    
    - 하나의 덩어리로 묶음
